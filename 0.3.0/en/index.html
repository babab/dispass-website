DisPass 0.3.0
The password manager that doesn't store your passwords

DisPass can be used as any traditional password manager, but has one key difference.
DisPass does not store your passwords anywhere, so you can never lose them.

This is the development version, documentation may not be up to date. For the latest stable release, see DisPass 0.2.0

No storage of passphrases

Instead it uses a label (something you can easily remember) to generate the same passphrase (for a website) everytime you want to login.

Strong passphrases

DisPass creates strong and unique passphrases formed from a master password and a label (and some optional parameters), helping you get rid of the bad habit of using a single password for multiple websites.

Command line or GUI

DisPass is written in Python and runs on GNU/Linux, *BSD, MacOS X and Windows. Dispass is a console application, but also has a simple graphical interface.

An interface for Emacs has been created by Tom Willemsen. Be sure to check it out if you use Emacs.
